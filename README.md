# 🩸 Blood Group Prediction System  

## 📌 Project Overview  
This project is a **Machine Learning-based Blood Group Prediction System** that predicts a person's blood group based on input features.  
It covers the complete data science workflow including **data preprocessing, analysis, model building, and evaluation**.

---

## 🚀 Features  
- ✔️ Data Cleaning & Preprocessing  
- ✔️ Exploratory Data Analysis (EDA)  
- ✔️ Feature Engineering  
- ✔️ Machine Learning Model Building  
- ✔️ Model Evaluation & Accuracy Check  
- ✔️ User Input Prediction System  

---

## 🛠️ Tech Stack  

**Programming Language:**  
- Python  

**Libraries Used:**  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- Seaborn  

---

## 📊 Project Workflow  

```
Data Collection → Data Cleaning → Data Analysis → Feature Selection → 
Model Training → Model Evaluation → Prediction
```
